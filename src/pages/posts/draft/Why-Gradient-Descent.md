# Why Gradient Descent

Last time, I wrote an introductory article on gradient descent and machine learning. In case you haven't read that article, machine learning is essentially just optimizing parameters on a loss function via gradient descent — a process finds the minimum points of a function (see my previous post to know more). Intuitively, we ask "Why gradient descent?" Apparently, gradient descent is not the only way to find minimum points, and we have learned plenty of ways to find minimum points throughout the study of mathematics. Therefore, there must be some underlying factors that make gradient descent the way we employ. Today, let's go through some optimization methods one by one; explore the ration behind using gradient descent.
